<span data-ttu-id="cef2b-121">Bedenken Sie folgende Punkte beachten Sie, wie Sie die Anwendung zum Arbeiten mit verschiedenen Versionen des EWS-Schemas entwerfen:</span><span class="sxs-lookup"><span data-stu-id="cef2b-121">Keep the following points in mind as you design your application to work with different versions of the EWS schema:</span></span>
  
- <span data-ttu-id="cef2b-122">Aktivieren Sie/deaktivieren Funktionen basierend auf die Schemaversion.</span><span class="sxs-lookup"><span data-stu-id="cef2b-122">Turn on/off functionality based on the schema version.</span></span> <span data-ttu-id="cef2b-123">Sie benötigen Clientfunktionen auf die Schemaversion und in einigen Fällen auf die Version des Diensts zuordnen möchten.</span><span class="sxs-lookup"><span data-stu-id="cef2b-123">You'll want to map client functionality to the schema version and, in some cases, to the version of the service.</span></span> <span data-ttu-id="cef2b-124">Das folgende Beispiel gibt die Version des Schemas und der Dienst einen [PropertySet](http://msdn.microsoft.com/en-us/library/office/microsoft.exchange.webservices.data.propertyset%28v=exchg.80%29.aspx) anhand zurück.</span><span class="sxs-lookup"><span data-stu-id="cef2b-124">The following example will return a [PropertySet](http://msdn.microsoft.com/en-us/library/office/microsoft.exchange.webservices.data.propertyset%28v=exchg.80%29.aspx) based on the version of the schema and service.</span></span> 
    
  ```cs
  private static PropertySet InitPropertySetByVersion(ExchangeService service)
  {
      PropertySet props;
      // The schema version to target to access the NormalizedBody property 
      // is Exchange2013 or later. The server version to target to access the 
      // NormalizedBody property on an email is 15 or later, which 
      // equates to Exchange 2013.
      if (service.RequestedServerVersion >= ExchangeVersion.Exchange2013 &amp;&amp;
          service.ServerInfo.MajorVersion >= 15)
      {
          props = new PropertySet(EmailMessageSchema.NormalizedBody);
      }
      else
      {
          props = new PropertySet(EmailMessageSchema.Body);
      }
      return props;
  }
  ```
