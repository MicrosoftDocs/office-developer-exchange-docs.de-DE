Öffentliche Ordner in Exchange verwenden ein spezielles Postfachdesign (anstelle einer Öffentliche Ordner-Datenbank), um die Hierarchie und Inhalte von öffentlichen Ordnern zu speichern. Public folder permissions are managed through Role Based Access Control (RBAC).</span></span>
  
<span data-ttu-id="057ae-p102">Clientzugriffstechnologien, wie Exchange-Webdienste (EWS) und die EWS Managed API ermöglichen den programmgesteuerten Zugriff auf die Öffentliche Ordner-Hierarchie und die Elemente in einer Öffentliche Ordner-Datenbank.
