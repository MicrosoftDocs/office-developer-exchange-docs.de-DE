# <a name="accuracy"></a>Genauigkeit

Das **Genauigkeits** Element gibt die Genauigkeit des breiten-und Längengrads der zugeordneten Postadresse an. 
  
```XML
<Accuracy></Accuracy>
```

 **xs: Double**
