# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

Das Element **AlternateMailboxes** stellt eine Sammlung von alternativen Postfächern. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
