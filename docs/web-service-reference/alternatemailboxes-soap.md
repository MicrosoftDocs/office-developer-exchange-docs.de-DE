# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

Das **AlternateMailboxes** -Element stellt eine Auflistung von alternativen Postfächern dar. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```
