# <a name="arrayofdomainresponse-soap"></a>ArrayOfDomainResponse (SOAP)

Das **ArrayOfDomainResponse** -Element enthält ein Array von Antworten für die Einstellungen jeder angeforderten Domäne. 
  
```XML
<ArrayOfDomainResponse>
   <DomainResponse/>
</ArrayOfDomainResponse>
```

 **ArrayOfDomainResponse**
