# <a name="arrayofdomainresponse-soap"></a>ArrayOfDomainResponse (SOAP)

Das **ArrayOfDomainResponse** -Element enthält ein Array von Antworten für jede angeforderte Domäne Einstellungen. 
  
```XML
<ArrayOfDomainResponse>
   <DomainResponse/>
</ArrayOfDomainResponse>
```

 **ArrayOfDomainResponse**
