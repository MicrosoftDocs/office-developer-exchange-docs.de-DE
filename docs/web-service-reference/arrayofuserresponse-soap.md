# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

Das **ArrayOfUserResponse** -Element enthält ein Array von Elementen [UserResponse (SOAP)](userresponse-soap.md) . 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```

 **ArrayOfUserResponse**
