# <a name="displayname-soap"></a>DisplayName (SOAP)

Das **DisplayName** -Element stellt einen Anzeigenamen ein. 
  
```XML
<DisplayName/>
```

 **string**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |Stellt eine alternative Postfach an.  <br/> |
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |Stellt Informationen zum Standort und Metadaten für ein Dokument sharing-Location.  <br/> |
   
## <a name="text-value"></a>Textwert

Der Textwert der **DisplayName** -Elements ist der Anzeigename einer Einstellung.
