# <a name="domain"></a>Domäne

Das **Domain** -Element identifiziert eine einzelne SMTP-Domäne. 
  
```xml
<Domain Name="" IncludeSubdomains="" />
```

 **StmpDomain**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

|**Attribut**|**Beschreibung**|
|:-----|:-----|
|Name  <br/> |Gibt den Namen einer Domäne an. Dieses Attribut ist erforderlich.  <br/> |
|IncludeSubdomains  <br/> |Gibt an, ob Unterdomänen der durch das **Name** -Attribut identifizierten Domäne als intern betrachtet werden. Dieses Attribut ist optional.  <br/> |
   
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[InternalDomains (SmtpDomainList)](internaldomains-smtpdomainlist.md) <br/> |Gibt die Liste der internen SMTP-Domänen der Organisation an.  <br/> |
   
## <a name="text-value"></a>Textwert

Keine.
  
## <a name="remarks"></a>Bemerkungen

Das Schema, das dieses Element beschreibt, befindet sich im virtuellen IIS-Verzeichnis, das Exchange-Webdienste hostet.
