# <a name="ecpurl-ret-pox"></a>EcpUrl-ret (POX)

Das Element **EcpUrl ret** gibt eine partielle URL, die kombiniert werden kann, mit dem [EcpUrl (POX)](ecpurl-pox.md) Elementwert generiert eine URL, die Einstellungen für einen e-Mail-aktivierten Benutzer Zugriff auf verwendet werden können.

```XML
<EcpUrl-ret/>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[Protokoll (POX)](protocol-pox.md) <br/> |Enthält die Spezifikationen für die Verbindung eines Clients mit dem Computer, auf der Microsoft Exchange Server ausgeführt wird, die die Clientzugriffs-Serverrolle installiert ist.  <br/> |
   
## <a name="text-value"></a>Textwert

Der Textwert stellt eine partielle URL, die kombiniert werden kann, mit dem [EcpUrl (POX)](ecpurl-pox.md) Elementwert generiert eine URL, die Einstellungen für den Benutzer Zugriff auf verwendet werden können. 
  
## <a name="remarks"></a>Hinweise

Das **EcpUrl ret** -Element ist ein optionales untergeordnetes Element des **Protokoll** -Elements.
