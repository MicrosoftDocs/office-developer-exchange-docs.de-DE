# <a name="extendedproperties-nonemptyarrayofextendedfielduris"></a>ExtendedProperties (NonEmptyArrayOfExtendedFieldURIs)

Das **ExtendedProperties** -Element enthält die erweiterten Eigenschaften, die für die einheitlichen Kontaktspeicher Vorgänge verwendet werden. 
  
```XML
<ExtendedProperties>
   <ExtendedProperty/>
</ExtendedProperties>
```

 **NonEmptyArrayOfExtendedFieldURIs**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

[Extended (pathtoextendedfieldtype Schematyp)](extendedproperty-pathtoextendedfieldtype.md)
  
### <a name="parent-elements"></a>Übergeordnete Elemente

[GetImItems](getimitems.md)  |  [GetImItemList](getimitemlist.md)
  
## <a name="remarks"></a>Bemerkungen

Dieses Element wurde in Exchange Server 2013 eingeführt.
  
Das Schema, das dieses Element beschreibt, befindet sich im virtuellen IIS-Verzeichnis, das Exchange-Webdienste hostet.
