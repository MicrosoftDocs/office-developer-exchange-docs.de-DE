# <a name="folderchanges"></a>FolderChanges

Das Element **FolderChanges** stellt eine Auflistung von Änderungen für einen Ordner. 
  
[UpdateFolder](updatefolder.md)
  
[FolderChanges](folderchanges.md)
  
```xml
<FolderChanges>
   <FolderChange/>
</FolderChanges>
```

 **NonEmptyArrayOfFolderChangesType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[FolderChange](folderchange.md) <br/> |Stellt eine Änderung in einem gemeinsamen Ordner ausgeführt werden.  <br/> |
   
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[UpdateFolder](updatefolder.md) <br/> |Stellt die Operation, die verwendet wird, um Eigenschaften für einen Ordner zu aktualisieren.  <br/> Es folgt der XPath-Ausdruck, der dieses Element:  <br/>  `/UpdateFolder` <br/> |
   
## <a name="remarks"></a>Hinweise

Das Schema, das dieses Element beschreibt, befindet sich im virtuellen EWS-Verzeichnis des Computers, der MicrosoftExchange Server 2007 mit installierter Clientzugriff-Serverrolle ausführt.
