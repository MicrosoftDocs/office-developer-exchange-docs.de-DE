# <a name="getappmanifests"></a>GetAppManifests

Das **GetAppManifests** -Element ist das Basiselement für eine Anforderung an das Manifest für apps zurückzugeben. 
  
```xml
<GetAppManifests />
```

 **GetAppManifestsType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

[ApiVersionSupported](apiversionsupported.md) | [SchemaVersionSupported](schemaversionsupported.md)
  
### <a name="parent-elements"></a>Übergeordnete Elemente

Keine.
  
## <a name="text-value"></a>Textwert

Keine.
  
## <a name="remarks"></a>Hinweise

Das **GetAppManifests** -Element ist für Clients, die Exchange Online und Versionen von Microsoft Exchange Server beginnend mit Exchange 2013 abzielen.
