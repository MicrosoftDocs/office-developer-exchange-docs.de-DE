# <a name="globaluniqueunreadsenders"></a>GlobalUniqueUnreadSenders

Das **GlobalUniqueUnreadSenders** -Element gibt eine Liste aller Personen, die Nachrichten gesendet wurden, die derzeit in dieser Unterhaltung ungelesen in allen Ordnern im Postfach sind.
