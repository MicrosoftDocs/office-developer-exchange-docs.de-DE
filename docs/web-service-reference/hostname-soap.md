# <a name="hostname-soap"></a>Hostname (SOAP)

Das **Hostname** -Element darstellt, der den hostnamensanteil der den vollständigen Namen des Computers. 
  
```XML
<Hostname/>
```

 **string**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[ProtocolConnection (SOAP)](protocolconnection-soap.md) <br/> |Stellt die Verbindung Protokoll des Webclients Server dar.  <br/> |
   
## <a name="text-value"></a>Textwert

Der Textwert der **Hostname** -Element ist die Hostnamenkomponente des den vollständigen Namen des Computers an.
