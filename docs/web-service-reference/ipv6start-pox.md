# <a name="ipv6start-pox"></a>IPv6Start (POX)

Das **IPv6Start** -Element identifiziert den Start eines Bereichs von IPv6-Adressen (IP Version 6), die zum Identifizieren eines Computers in einem Netzwerk verwendet werden.

[IPv6Start (POX)](ipv6start-pox.md)
  
```xml
<IPv6Start/>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[NetworkRequirements (POX)](networkrequirements-pox.md) <br/> |Enthält die Kriterien, die verwendet werden, um zu bestimmen, ob sich der Clientcomputer in einem Netzwerk befindet, das die Anforderungen des Internet Dienstanbieters erfüllt, um eine Verbindung mit dem Server herzustellen.  <br/> |
   
## <a name="text-value"></a>Textwert

Der Wert Text stellt eine IPv6-Adresse dar.
