# <a name="mailtipsaccessenabled-soap"></a>MailTipsAccessEnabled (SOAP)

Das **MailTipsAccessEnabled** -Element stellt das **MailTipsAccessEnabled ()** -Flag dar. Das **MailTipsAccessEnabled** -Element ist nur für die interne Verwendung. Dieses Element wird nicht von Clients verwendet. 
  
```XML
<MailTipsAccessEnabled>true | false</ MailTipsAccessEnabled>
```

 **Boolescher Wert**
