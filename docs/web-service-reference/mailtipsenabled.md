# <a name="mailtipsenabled"></a>MailTipsEnabled

Das **MailTipsEnabled** -Element gibt an, ob der Mail-Tipps Dienst verfügbar ist. 
  
```xml
<MailTipsEnabled>true | false</MailTipsEnabled>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[MailTipsConfiguration (MailTipsServiceConfiguration)](mailtipsconfiguration-mailtipsserviceconfiguration.md) <br/> |Enthält Konfigurationsinformationen für den e-Mail-Dienst Tipps Service.  <br/> |
   
## <a name="text-value"></a>Textwert

Der Textwert der dieses Element ist **true** , wenn der Mail-Tipps Dienst verfügbar ist. Der Wert ist **false** , wenn der Mail-Tipps-Dienst nicht verfügbar ist. 
  
## <a name="remarks"></a>Hinweise

Das Schema, das dieses Element beschreibt, befindet sich im virtuellen IIS-Verzeichnis, das Exchange-Webdienste hostet.
