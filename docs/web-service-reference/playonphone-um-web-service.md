# <a name="playonphone-um-web-service"></a>PlayOnPhone (UM-Webdienst)

Das **PlayOnPhone** -Element definiert eine Anforderung an ein Element auf ein Telefon wiedergegeben werden sollen. 
  
[PlayOnPhone (UM-Webdienst)](playonphone-um-web-service.md)
  
```xml
<PlayOnPhone>
  <entryId>   </entryId>
  <DialString>   </DialString>
</PlayOnPhone>
```

 **complexType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[EntryId (UM-Webdienst)](entryid-um-web-service.md) <br/> |Enthält den Wert, der den Bezeichner des Elements, auf das Telefon in einer Anforderung [PlayOnPhone-Vorgang (UM-Webdienst)](playonphone-operation-um-web-service.md) abzuspielen darstellt.  <br/> |
|[DialString (UM-Webdienst)](dialstring-um-web-service.md) <br/> |Enthält den Wert für die Rufnummer gewählt.  <br/> |
   
### <a name="parent-elements"></a>Übergeordnete Elemente

Keine.
  
## <a name="text-value"></a>Textwert

Keine.
