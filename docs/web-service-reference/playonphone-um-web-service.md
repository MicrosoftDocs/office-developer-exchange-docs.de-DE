# <a name="playonphone-um-web-service"></a>PlayOnPhone (um-Webdienst)

Das **PlayOnPhone** -Element definiert eine Anforderung, ein Element auf einem Telefon wiederzugeben. 
  
[PlayOnPhone (um-Webdienst)](playonphone-um-web-service.md)
  
```xml
<PlayOnPhone>
  <entryId>   </entryId>
  <DialString>   </DialString>
</PlayOnPhone>
```

 **complexType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

|**Element**|**Beschreibung**|
|:-----|:-----|
|[Eingabe-Nr (um-Webdienst)](entryid-um-web-service.md) <br/> |Enthält den Wert, der den Bezeichner des Elements darstellt, das in einer [PlayOnPhone-Vorgangsanforderung (um-Webdienst)](playonphone-operation-um-web-service.md) auf dem Telefon wiedergegeben werden soll.  <br/> |
|[Wähl Dienst (um-Webdienst)](dialstring-um-web-service.md) <br/> |Enthält den Wert für die Telefonnummer, die gewählt werden soll.  <br/> |
   
### <a name="parent-elements"></a>Übergeordnete Elemente

Keine.
  
## <a name="text-value"></a>Textwert

Keine.
