# <a name="proposedstart-meetingregistrationresponseobjecttype"></a>ProposedStart (MeetingRegistrationResponseObjectType)

Das **ProposedStart (MeetingRegistrationResponseObjectType)** -Element gibt einen Teilnehmer vorgeschlagene neue Startzeit für eine Besprechung. 
  
```XML
<ProposedStart />
```

 **dateTime**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

[AcceptItem](acceptitem.md) | [TentativelyAcceptItem](tentativelyacceptitem.md) | [DeclineItem](declineitem.md)
  
## <a name="text-value"></a>Textwert

Der Textwert des **ProposedStart (MeetingRegistrationResponseObjectType)** -Elements ist das vorgeschlagener Anfangstermin und die Uhrzeit der Besprechung. 
  
## <a name="remarks"></a>Hinweise

Dieses Element wurde in Exchange Server 2013 Service Pack 1 (SP1) eingeführt.
  
Das Schema, das dieses Element beschreibt, befindet sich im virtuellen IIS-Verzeichnis, das Exchange-Webdienste hostet.
