# <a name="proposenewtime"></a>ProposeNewTime

Das **ProposeNewTime** -Element gibt eine Antwortobjekt, der angibt, dass der Besprechungsteilnehmer eine neue Besprechungszeit vorschlagen kann. 
  
```XML
<ProposeNewTime ObjectName=""></ProposeNewTime>
```

 **ProposeNewTimeType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

****

|**Attribut**|**Beschreibung**|
|:-----|:-----|
|Objektname  <br/> |Der Name der im Response-Objekt.  <br/> |
   
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

[ResponseObjects](responseobjects.md)
  
## <a name="remarks"></a>Hinweise

Dieses Element wurde in Exchange Server 2013 Service Pack 1 (SP1) eingeführt.
  
Das Schema, das dieses Element beschreibt, befindet sich im virtuellen IIS-Verzeichnis, das Exchange-Webdienste hostet.
