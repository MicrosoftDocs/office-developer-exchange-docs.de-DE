# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

Das **ProtocolConnections** -Element enthält null oder mehr Protokoll Verbindungen. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```
