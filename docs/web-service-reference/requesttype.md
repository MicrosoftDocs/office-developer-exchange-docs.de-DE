# <a name="requesttype"></a>RequestType

Das **RequestType** -Element gibt an, ob es sich bei einer Proxyanforderung um eine Cross-Site-oder eine gesamtstrukturübergreifende Anforderung handelt. 
  
```xml
<RequestType>CrossSite or CrossForest</RequestType>
```

 **AvailabilityProxyRequestType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
  
### <a name="attributes"></a>Attribute

Keine.
  
### <a name="child-elements"></a>Untergeordnete Elemente

Keine.
  
### <a name="parent-elements"></a>Übergeordnete Elemente

Dieses Element verfügt nicht über ein übergeordnetes Objekt im Schema. Dieses Element wird im SOAP-Header verwendet. Weitere Informationen zur Verwendung dieses Elements finden Sie in der WSDL-Datei.
  
## <a name="text-value"></a>Textwert

Für dieses Element ist ein Textwert erforderlich. Im Folgenden sind die möglichen Werte aufgeführt:
  
- CrossSite
    
- CrossForest
