# <a name="stringattributedvalue"></a>StringAttributedValue

Das **StringAttributedValue** -Element gibt eine Instanz in einem Array von Attributen an, die mit einem Persona-Element verknüpft sind. 
  
```XML
<StringAttributedValue>
   <Value/>
   <Attributions/>
</StringAttributedValue>
```

 **StringAttributedValueType**
## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete und übergeordnete Elemente erläutert.
